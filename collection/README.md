# Collection Workspace

This directory holds Silver Spring Records collection-related data and strategy artifacts.

## Data Files

- `music-collection.json` - public collection table source for `collection.html`.
- `music-wishlist.json` - public wishlist table source for collection tracks that were not safely added to the crate.
- `spotify-playlist.json` - Spotify playlist export/source data.
- `spotify-purchase-log.json` - purchase history tracking.
- `spotify-purchase-candidates.json` - Spotify candidate tracks for purchase review.
- `playlist-purchase-candidates.json` - playlist-derived purchase candidates.
- `bpm-supreme-miri-jack-links.json` - BPM Supreme matching/link research.
- `apple-music-open-tabs-consolidated.json` - deduplicated songs extracted from open Apple Music playlist tabs.
- `bpm-supreme-apple-music-search-results.json` - restartable BPM Supreme search ledger, including misses.
- `bpm-supreme-batch-runbook.md` - required pacing, checkpoint, skip, and crate-add criteria for batch runs.

## Strategy Notes

Keep collection planning, purchase strategy, sourcing notes, and review docs in this directory so the collection workflow stays separate from the public site pages.
